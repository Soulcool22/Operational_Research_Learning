# 运筹学优化算法学习工具包 - 项目规则

## 🎯 项目概述

这是一个专为运筹学学习设计的Python项目，包含线性规划、整数规划、运输问题、网络流优化等核心算法的演示和可视化。项目旨在通过实际案例和图形化展示，帮助初学者理解和掌握运筹学的核心概念。

## 📁 项目结构理解

### 核心文件
- `main_demo.py` - 主程序入口，提供交互式菜单
- `font_config.py` - 中文字体配置，确保图表正确显示中文
- `requirements.txt` - 项目依赖包列表

### 演示模块
- `linear_programming_demo.py` - 线性规划演示（生产计划优化）
- `integer_programming_demo.py` - 整数规划演示（设施选址问题）
- `transportation_problem_demo.py` - 运输问题演示（供应链优化）
- `network_flow_demo.py` - 网络流优化演示（最大流问题）
- `large_scale_optimization.py` - 大规模优化演示
- `visualization_demo.py` - 可视化专项演示

### 生成文件
- `*.png` - 各种可视化图表（11个图片文件）
- `题目集/` - 详细的题目描述和学习指南

## 🔧 代码规范

### Python编码规范
1. **编码声明**：所有Python文件必须包含UTF-8编码声明
   ```python
   #!/usr/bin/env python
   # -*- coding: utf-8 -*-
   ```

2. **中文支持**：
   - 导入并使用 `font_config.setup_chinese_font()`
   - 确保所有图表标签正确显示中文
   - 使用LaTeX格式表示数学角标：`$x_1$` 而不是 `x₁`

3. **模块结构**：
   - 每个演示模块包含一个主类
   - 类名使用驼峰命名法（如 `LinearProgrammingDemo`）
   - 包含 `main()` 方法作为模块入口

### 数学建模规范
1. **变量命名**：使用有意义的变量名
2. **约束条件**：清晰注释每个约束的含义
3. **目标函数**：明确标注优化方向（最大化/最小化）

## 📊 可视化规范

### 图表要求
1. **中文字体**：必须正确显示中文标签和标题
2. **图表尺寸**：统一使用合适的figsize
3. **保存格式**：PNG格式，DPI=300（高质量）
4. **文件命名**：使用描述性的英文文件名

### 样式规范
1. **颜色方案**：使用seaborn-v0_8样式
2. **图例位置**：合理放置，不遮挡主要内容
3. **网格线**：适当使用，透明度alpha=0.3
4. **标题格式**：使用fontweight='bold'
