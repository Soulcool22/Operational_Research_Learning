# 可视化分析 - 优化算法过程演示

## 📋 问题背景

运筹学中的优化算法往往比较抽象，初学者难以直观理解算法的工作原理和求解过程。通过可视化技术，我们可以将抽象的数学概念转化为直观的图形展示，帮助学习者更好地理解线性规划可行域、梯度下降过程、网络流分布、敏感性分析等核心概念。

## 🎯 问题描述

### 基本情况
- **可视化对象**：多种运筹学优化问题和算法
- **展示维度**：2D平面图、3D立体图、动态过程图
- **教学目标**：通过图形化展示增强理解
- **技术手段**：Python + Matplotlib + Seaborn + NetworkX

### 可视化内容
需要创建多种类型的可视化图表，展示不同优化问题的特征和求解过程。

## 📊 可视化任务清单

### 1. 线性规划可行域可视化
#### 简单二维线性规划
**问题设置**：
- 目标函数：$\max 3x_1 + 2x_2$
- 约束条件：
  - $2x_1 + 3x_2 \leq 12$
  - $x_1 + 2x_2 \leq 8$  
  - $x_1, x_2 \geq 0$

**可视化要求**：
- [ ] 绘制约束线
- [ ] 标识可行域（阴影区域）
- [ ] 标出最优解点
- [ ] 显示目标函数等值线
- [ ] 添加中文标签和说明

#### 复杂约束可行域
**问题设置**：
- 5个随机生成的线性约束
- 展示更复杂的可行域形状

**可视化要求**：
- [ ] 多约束线的交互
- [ ] 可行域的几何形状
- [ ] 约束方程的标注

### 2. 优化过程动态演示

**问题设置**：

- 初始点：$(0, 0)$
- 学习率：$\alpha = 0.1$

**可视化要求**：
- [ ] 绘制目标函数等高线
- [ ] 显示迭代路径
- [ ] 标注每次迭代点
- [ ] 显示梯度方向

### 3. 三维优化表面

**函数类型**：

2. **Ackley函数**：多峰函数

**可视化要求**：

- [ ] 约束边界显示
